Hi, This is because your container didn't provide bean manager in default JNDI name: java:comp/env/BeanManager
You have to figure out which JNDI name that your container is using, and declare a library property setting(org.zkoss.zkplus.cdi.beanManager.jndiName) in zk.xml:

<library-property>
    <name>org.zkoss.zkplus.cdi.beanManager.jndiName</name>
    <value>{your_container_cdi_bean_manager_jndi_name}</value>
</library-property>

Jease 2.6 released - The ZK-based CMS without headaches

mjablonskiTop Contributor
9 Jan 2012 08:12:31 GMT
9 Jan 2012 08:12:31 GMT

Hi,

I'm happy to announce the release of Jease 2.6 which is powered by the recently released ZK 5.0.10.

Jease is a Java based Content Management System which uses ZK in two ways:

- ZK-Richlets to provide a powerful and stable GUI technology for the backend.
- ZUML as scripting technology to enhance the public view of your site.

The main focus of Jease 2.6 is the automatic HTML to PDF conversion for content objects. Just check out the link at the bottom of jease.org to see how it works.

You can find the full release note and download here:

http://www.jease.org/download/2.6/

If you want to give it a test drive, just use the demo:

http://www.jease.org/demo/

Thanks to the ZK community for their fantastic work, without ZK a project like Jease wouldn't be that (j)easy...:-)

Cheers, Maik

How to properly append items to a collection when using form binding?

billyboingo
5 Sep 2012 16:54:29 GMT
5 Sep 2012 16:54:29 GMT

So let's say I have a person POJO:

public class Person  {

private Set<Phone> phoneNumbers = new HashSet<Phone>();

...

public void setPhoneNumbers(final Set<Phone> numbers){
   this.phoneNumbers = numbers;
}

public Set<Phone> getPhoneNumbers(){
  return this.phoneNumbers;
}

...

...and the Phone class: 

public class Phone  {

private String type;
private String number;

...

//Getters and setters

...

In my viewmodel, I have two methods for adding phone numbers from a person:

public class PersonVM  {

...

    @Command
    public void addPhoneNumber(@BindingParam("fx") final Form form) {
        Collection<Phone> numbers = (Collection<Phone>) form
                .getField("phoneNumbers");
        numbers.add(new Phone());
        BindUtils.postNotifyChange(null, null, form, "phoneNumbers");
    }

...

Then in the zul:

                          <row>
                                <cell colspan="6">
                                    <button label="Add"
                                        onClick="@command('addPhoneNumber',fx=fx)">
                                    </button>
                                    <listbox
                                        model="@bind(fx.phoneNumbers)">
                                        <template name="model">
                                            <listitem>
                                                <listcell hflex="1">
                                                    <combobox
                                                        model="@load(vm.phoneTypes)" hflex="true"
                                                        selectedItem="@bind(each.type)" readonly="true">
                                                        <template
                                                            name="model">
                                                            <comboitem
                                                                label="@load(each)"/>
                                                        </template>
                                                    </combobox>
                                                </listcell>
                                                <listcell hflex="2">
                                                    <textbox
                                                        value="@bind(each.number)">
                                                    </textbox>
                                                </listcell>
                                            </listitem>
                                        </template>
                                    </listbox>
                                </cell>
                            </row>

This works fine...except that it doesn't cause the form status to flip to dirty. Any ideas on what I need to change to have changes to the collection and/or the contents of the collection properly flip the dirty flag?

This is important because I have toolbar buttons that react to the form state...

Thanks in advance!
